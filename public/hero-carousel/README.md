# Fotos del carrusel de la Hero

Coloca aquí tus imágenes para el carrusel principal.

## Nombres esperados por la app

- `slide-1.jpg`
- `slide-2.jpg`
- `slide-3.jpg`

## Logotipos tipográficos (hero)

- `logo-blanco.png`
- `logo-negro.png`

La hero intentará mostrar uno de estos logos sobre el carrusel.
Si no existen, mostrará automáticamente el título en texto como respaldo.

## Recomendación

- Formato: JPG o WEBP
- Tamaño sugerido: 1920x1080
- Peso recomendado: menor a 500 KB por imagen

Si quieres usar más o menos imágenes, puedo ajustarte el arreglo `HERO_SLIDES` en `src/App.tsx`.
